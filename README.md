# 📈 Indian Stock Market Analysis

## 📌 Project Overview
End-to-end analysis of top 5 Indian stocks (Reliance, TCS, Infosys, 
HDFC Bank, Wipro) using real NSE market data — covering data cleaning, 
statistical analysis, risk evaluation, and interactive visualizations.

## 🛠️ Tools & Libraries
- Python
- Pandas — Data Cleaning & Manipulation
- NumPy — Statistical Calculations
- Matplotlib & Seaborn — Static Visualizations
- Plotly — Interactive Dashboards
- yfinance — Real-time Stock Data API

## 📂 Project Structure
- `data/` — Raw and cleaned stock data (CSV)
- `notebooks/` — Step-by-step analysis notebooks
- `visuals/` — Charts and interactive HTML dashboards

## 📊 Key Analysis Performed
1. **Data Cleaning** — Handled missing values and formatting issues
2. **Daily Returns & Volatility** — Risk measurement using NumPy
3. **Correlation Analysis** — Heatmap showing stock relationships
4. **Moving Averages** — 50-day & 200-day trend analysis
5. **Performance Comparison** — Best vs worst performing stocks (2024)
6. **Risk vs Return Analysis** — Scatter plot for risk profiling
7. **Interactive Candlestick Chart** — Plotly-based price visualization
8. **Price vs Volume Dashboard** — Combined interactive analysis
